- ## 期末專題報告
    - #### 如何在 localhost 安裝與測試之詳細步驟
        首先clone repo
        ```
        git clone https://github.com/Alx-Lai/wp1092.git
        ```
        然後install 該有的東西
        ```
        cd wp1092/final
        cd frontend
        npm install
        cd ../backend
        npm install
        ```
        開兩個command line 一個在frontend 一個在backend
        ```
        //frontend
        npm start
        //backend
        npm run server
        // 等到 顯示 “Mongo database connected!” 再開始遊戲
        ```
